10) Show the output produced by each of the following program fragments.
Assume that i and j are int variables.

a)

```c
i = 1;
j = ++i * 3 - 2;
printf("%d %d", i, j);
```
=> 2 4

b)

```c
i = 5;
j = 3 - 2 * i++;
printf("%d %d", i, j);
```
=> 6 -7

c)
```c
i = 7;
j = 3 * i-- + 2;
printf("%d %d", i, j);
```

=> 6 23

d)

```c
i = 7;
j = 3 + --i * 2;
printf("%d %d", i, j);
```
=> 6 15
